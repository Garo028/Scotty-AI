# Scotty AI - Open Source

Your AI brain - same tech as the real Lumora backend.

## Setup (2 minutes)

### 1. Get FREE API Key (choose one)

**Option A: Groq (FASTEST)**
- Go to https://console.groq.com
- Sign up (free)
- API Keys → Create Key
- Copy the key

**Option B: OpenRouter**
- Go to https://openrouter.ai
- Sign up (free $5 credit)
- Keys → Create Key
- Copy the key

### 2. Add to config.env
```
GROQ_API_KEY=your-key-here
```
OR
```
OPENROUTER_API_KEY=your-key-here
```

### 3. Run
```bash
cd backend
npm install
npm start
```

### 4. Open index.html
Chat with your AI!

---

## What's Inside

- **index.html** - Chat UI. Just open in browser
